盒模型與 box-sizing

【題目需求】
  • 認巢狀四層；同一組 width／padding／border，比兩種 box-sizing 誰比較寬。
【執行結果】
margin
border
padding
content
border-box
content
content-box
content

【學習重點】

  • padding／margin 簡寫順序：上 → 右 → 下 → 左。
  • 置中：margin-inline: auto（或 margin: 0 auto）。
  • box-sizing：對照本頁兩盒虛線；細節見延伸 MDN。
