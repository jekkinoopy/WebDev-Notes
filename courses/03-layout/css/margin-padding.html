盒模型與 box-sizing

【題目需求】
  • 辨識由外而內 margin、border、padding、content 的層級。
  • 在相同 width、height、padding、border 下，比較 content-box 與 border-box 的總佔位。

【學習重點】

  • padding：增加邊框內側與內容之間的距離。
  • 四邊簡寫：順序為「上、右、下、左」；兩個值時為「上下、左右」。
  • margin：在邊框外推開與其他盒子的距離；可為負值，百分比相對於包含塊寬度。
  • 水平置中：區塊在一般排版流中，設寬度後可使用 margin-inline: auto；亦常寫成 margin: 0 auto。
  • content-box：所設 width／height 多指內容區；padding、border 會往外疊加，總佔位變大。
  • border-box：所設 width／height 含 padding 與 border；內容區會向內縮，總外框較易預估。
