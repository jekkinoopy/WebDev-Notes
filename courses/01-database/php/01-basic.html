PHP 基礎：變數解析與交換實戰

掌握單雙引號對變數解析的差異，並透過經典「三杯水」演算法實作變數對調。

變數：單雙引號對照

$a = 10;
$b = 50;
echo '$a=' . $a; // 顯示: $a=10
echo "$b=" . $b; // 顯示: 50=50
$a=10
50=50

變數交換練習：三杯水交換法

$temp = $a;   // 備份 A 到 Temp
$a = $b;      // 把 B 的值給 A
$b = $temp;   // 把 Temp 備份還原給 B

echo '$a=' . $a;
echo '$b=' . $b;
$a=50
$b=10
