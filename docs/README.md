# 說明文件索引（`docs/`）

根目錄還有一些與課程並排的檔案，容易眼花；**協作／設計相關**可從這裡進。

| 檔案 | 用途 |
|------|------|
| [ai-collab-guide.md](ai-collab-guide.md) | 與 AI／Cursor 協作習慣、怎麼下需求、成長備忘（**非**正式公司規範） |
| [colors-quickstart.md](colors-quickstart.md) | 色彩速查（改色步驟、快速配色片段） |
| [../CURSOR-HANDOFF.md](../CURSOR-HANDOFF.md) | 接力必讀、協作邊界、〈變更〉紀錄 |
| [../COLOR-GUIDE.md](../COLOR-GUIDE.md) | 色彩完整指南（與 `colors-config.css` 搭配） |

## 根目錄上還常看到的

| 路徑 | 建議心裡分類 |
|------|----------------|
| `README.md` | **專案總覽**（入口連結表；勿再整份貼色彩長文，避免與本索引混淆） |
| `colors-preview.html` | 色票預覽頁 |
| `index.html` | 站台／導覽入口 |
| `temp/` | 臨時盤點、草稿；專題結束可刪或自己歸檔 |
| `原檔/` | 個人原稿備份，非課程正式入口 |

之後若要把 `COLOR-GUIDE.md` 移進 `docs/`，記得全域搜尋舊連結再改一次即可。
